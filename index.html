Most recent work

RISC-V Processor with Pipeline Architecture

This project involves the design and implementation of a 32-bit RISC-V processor featuring a multi-stage pipelined architecture. The processor was developed using SystemVerilog and adheres to the RISC-V instruction set architecture (ISA), enabling efficient execution of instructions through concurrent operation in different pipeline stages.

GitHub Repository

Blockchain In Python

This project is a Python-based blockchain implementation that simulates the core functionalities of a blockchain. It includes features like mining new blocks, handling transactions, and maintaining a chain of blocks. The Flask web framework is used to provide APIs for interacting with the blockchain, such as adding transactions, mining blocks, and displaying the full chain.

GitHub Repository

Automatic Email Generator in Python

This script automates sending personalized emails to contacts listed in a file and retrieves the last 10 sent email details from the Gmail "Sent Mail" folder.

Safebox with Fingerprint sensor and Bluetooth module
